radicle-httpd

Read-only JSON API for your local Radicle node. The dashboard at personal.localhost reads from these endpoints.

Endpoints

  • /api/v1 Welcome / discovery (lists all available routes)
  • /api/v1/node This node’s identity, alias, peers and config
  • /api/v1/stats Total repository count and other counters
  • /api/v1/repos?show=all All repositories the node is replicating
  • /api/v1/repos?show=pinned Repositories listed in web.pinned.repositories inside $RAD_HOME/config.json (often ~/.radicle/config.json)
  • /api/v1/repos/<rid> Single repository by RID, e.g. rad:zXxx… (works even if /repos is empty)

Try it from the shell

curl -s http://api.localhost/api/v1/stats | jq
curl -s 'http://api.localhost/api/v1/repos?show=all' | jq '.[].payloads."xyz.radicle.project".data.name'
Browsers display raw JSON without much styling. For pretty formatting in Chrome/Brave/Edge install a JSON viewer extension, or pipe through jq.
